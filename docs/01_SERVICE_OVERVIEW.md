# 서비스 개요 (SERVICE OVERVIEW)

> **MBTI 투자 캐릭터 생성기** - 성격 유형으로 찾는 나만의 투자 스타일

---

## 🎯 핵심 가치 제안

### 문제 (Problem)
- 주식 투자 서비스는 초보자에게 **진입 장벽이 높음** (숫자, 차트, 전문 용어)
- 투자 정보는 많지만, **자기 성향과 연결해 이해할 수 있는 설명**은 부족
- MBTI는 대중적이지만 투자 영역과 연결된 콘텐츠는 거의 없음

### 솔루션 (Solution)
- MBTI 성격 유형을 투자 성향으로 재해석
- **은유 기반 설명**으로 복잡한 투자 개념을 쉽게 전달
- 캐릭터화된 결과로 재미와 공유 가능성 극대화

### 차별점 (Differentiation)
- ✅ 주식 종목을 **정보가 아닌 캐릭터 맥락**으로 설명
- ✅ 은유적 표현으로 개인화된 설명 제공
- ✅ SNS 공유에 최적화된 모바일 앱 스타일 UI

```
니즈:

- "어려운 용어 말고 쉽게 설명해줬으면..."
- "친구들과 비교해보고 싶어"
```

```
특징:

- "다른 유형도 궁금해"
```

## 🎨 서비스 특징

### 1. MBTI 기반 투자 성향 분석
- 16개 MBTI 유형별 투자 스타일 정의
- 리스크 성향, 의사결정 방식, 선호 투자 방식 분석

### 2. 5개 맞춤 테마 제공
각 MBTI별로 5개의 독특한 투자 테마 제공 (총 80개)

**예시 (INTJ):**
- 🏰 불장은 온다... 존버의 달인을 위한
- 🔬 데이터가 말해주는 숨겨진 진주
- 🎯 계획대로 되고 있어... 시스템 트레이딩
- 🧠 미래를 읽는 자, 혁신 기술주
- 📊 리스크는 계산했다, 분산 포트폴리오

### 3. 은유 기반 종목 설명
```
기존 방식:
"삼성전자는 PER 15배, PBR 1.2배입니다."

우리 방식:
"당신의 INTJ 성향에 맞는 '천천히 쌓이는 성' 같은 종목이에요.
단기 변동에 흔들리지 않고 장기적으로 견고하게 성장하는 스타일입니다."
```

### 4. 가상 포트폴리오 + MBTI 코멘트
- 초기 자산: 10,000,000원
- 수익률에 따른 MBTI별 코멘트
  - 상승장: "가즈아~!" (ENFP) / "계획대로" (INTJ)
  - 하락장: "패닉은 금물" (ISTJ) / "데이터 재점검" (INTJ)

## 🚀 사용자 여정

### 첫 방문자 플로우
```
1. 랜딩페이지 도착 (SNS 링크)
   ↓
2. 서비스 소개 확인
   ↓
3. [시작하기] 버튼 클릭
   ↓
4. MBTI 선택 (16개 버튼)
   ↓
5. 로딩 화면 (3초)
   ↓
6. 메인페이지 진입
   - 가상 자산 카드 확인
   - 5개 테마 스와이프 탐색
   - 각 테마별 10개 종목 확인
   ↓
7. 종목 클릭 → 상세 모달
   - 은유 설명 읽기
   - MBTI 성향 연결 이해
```

### 재방문자 플로우
```
1. 메인페이지 직접 접속 (북마크)
   ↓
2. LocalStorage에서 MBTI 로드
   ↓
3. 바로 메인페이지 표시
```

## 📊 성공 지표 (KPI)

### 사용자 지표
- **완료율**: 70% 이상 (MBTI 선택 → 결과 확인)
- **평균 체류 시간**: 3분 이상
- **재방문율**: 30% 이상

### 기술 지표
- **페이지 로딩 속도**: < 3초
- **로딩 화면**: < 3초
- **에러율**: < 1%

## 🎯 MVP 범위

### ✅ Phase 1 (현재 목표)
**UI/UX: 전체 페이지 구조 완성**
- 랜딩페이지
- MBTI 선택 화면
- 로딩 화면
- 메인페이지 (가상 자산 + 테마 + 종목)
- 종목 상세 모달
- 커뮤니티/자산현황 (목업)
- 설정 페이지

**기능: 핵심 기능만 구현**
- MBTI 선택 및 LocalStorage 저장
- 가상 자산 표시 + MBTI 코멘트
- 5개 테마 스와이프
- 테마별 10개 종목 리스트
- 종목 상세 모달

**데이터: 정적 JSON**
- 사전 정의된 16개 MBTI 프로필
- 80개 테마 데이터
- 종목 데이터 (정적)

### ⏸️ Phase 2 (향후)
- AI 캐릭터 생성 (LLM 연동)
- 실시간 주가 데이터
- 북마크 기능
- 커뮤니티 실제 구현
- 가상 포트폴리오 관리
