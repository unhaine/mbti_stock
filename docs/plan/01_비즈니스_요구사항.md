## 1. 문제 정의

### 1.1 배경 (Background)

#### 주식 투자 시장의 현황
- 2020년 이후 개인 투자자 급증 ("동학개미운동")
- MZ세대의 투자 관심도 증가
- 하지만 **초보자 이탈률 여전히 높음**

#### 기존 투자 서비스의 문제점
```
❌ 진입 장벽이 높음
   - 전문 용어 (PER, PBR, ROE 등)
   - 복잡한 차트와 지표
   - 숫자 중심의 정보 전달

❌ 개인화 부족
   - 모든 사용자에게 동일한 정보 제공
   - 개인의 성향과 무관한 추천

❌ 재미 요소 부재
   - 딱딱한 금융 앱 느낌
   - 학습 동기 부여 어려움
```

### 1.2 문제 진술 (Problem Statement)

> **"투자 정보는 넘쳐나지만, 초보자가 자신의 성향과 연결해 이해할 수 있는 방법은 부족하다."**

#### 구체적 문제
1. **인지적 부담**: 투자 = 어렵고 복잡한 것이라는 선입견
2. **연결성 부재**: 내 성향과 투자 스타일의 연결고리 부족
3. **흥미 저하**: 숫자와 차트만으로는 지속적 관심 유지 어려움

### 1.3 기회 (Opportunity)

#### MBTI의 대중화
- 국내 MBTI 인지도: **90% 이상** (20-30대 기준)
- "MBTI로 알아보는 ○○" 콘텐츠 인기
- 자기 이해 도구로 광범위하게 활용

#### AI 기술의 발전
- LLM을 활용한 개인화된 설명 생성 가능
- 복잡한 개념을 은유로 변환하는 능력
- 비용 효율적인 API 제공 (OpenAI, Anthropic)

#### 엔터테인먼트형 금융 서비스 트렌드
- 토스, 뱅크샐러드 등 친근한 금융 UX
- 게임화(Gamification) 요소 도입
- 밈(Meme) 문화와 금융의 결합

## 2. 타겟 사용자 분석

### 2.1 주요 페르소나

#### 페르소나 1: "투자 입문자 지민" (Primary)
```
👤 기본 정보
- 나이: 27세
- 직업: 스타트업 마케터
- 거주지: 서울

💭 특징
- MBTI에 관심 많고 친구들과 자주 공유
- 주식 투자에 관심은 있지만 어디서부터 시작할지 모름
- 유튜브로 투자 공부했지만 실전은 두려움
- SNS 활동 활발 (인스타그램, 카카오톡)

🎯 니즈
- "내 성향에 맞는 투자 방법이 뭘까?"
- "어려운 용어 말고 쉽게 설명해줬으면..."
- "친구들과 비교해보고 싶어"

😫 페인 포인트
- 투자 앱 켜면 무슨 말인지 모르겠음
- 손실 보면 멘탈 관리 어려움
- 혼자 공부하기 지루함
```

#### 페르소나 2: "호기심 많은 수현" (Secondary)
```
👤 기본 정보
- 나이: 24세
- 직업: 대학생 (경영학과)
- 거주지: 부산

💭 특징
- MBTI 테스트를 여러 번 해봄
- 주식은 해본 적 없지만 재테크에 관심
- 심리 테스트, 성향 분석 콘텐츠 좋아함
- 재미있으면 바로 공유하는 스타일

🎯 니즈
- "가볍게 해볼 수 있는 투자 관련 콘텐츠"
- "결과를 친구들에게 자랑하고 싶어"
- "다른 유형도 궁금해"

😫 페인 포인트
- 진지한 투자 공부는 부담스러움
- 실제 돈 넣기는 무서움
- 재미없으면 금방 이탈
```

### 2.2 타겟 세그먼트

| 세그먼트 | 비율 | 특징 | 우선순위 |
|---------|------|------|----------|
| **MBTI 팬 + 투자 초보** | 40% | 투자 입문 동기 부여 필요 | 🔴 High |
| **심리 테스트 애호가** | 30% | 엔터테인먼트 소비, 공유 | 🟡 Medium |
| **투자 경험자** | 20% | 재미로 해보는 사용자 | 🟢 Low |
| **기타** | 10% | 호기심 방문자 | 🟢 Low |

## 3. 사용자 니즈 분석

### 3.1 기능적 니즈 (Functional Needs)

#### Must Have (필수)
- ✅ MBTI 유형 선택 기능
- ✅ 투자 캐릭터 생성 및 설명
- ✅ 주식 추천 (최소 3개)
- ✅ 결과 확인 페이지

#### Should Have (중요)
- ✅ 결과 이미지 다운로드/공유
- ✅ 다른 MBTI 유형 탐색
- ✅ 추천 이유 상세 설명

#### Could Have (선택)
- ⭕ 투자 경험 입력 (추가 개인화)
- ⭕ 친구와 비교 기능
- ⭕ 캐릭터 재생성 (다른 결과 보기)

#### Won't Have (제외)
- ❌ 실시간 주가 정보
- ❌ 실제 매매 기능
- ❌ 포트폴리오 관리

### 3.2 감성적 니즈 (Emotional Needs)

#### 재미 (Fun)
- 심리 테스트하는 느낌
- 예상치 못한 은유적 표현
- 친구들과 비교하며 웃을 수 있는 요소

#### 자기 이해 (Self-Discovery)
- "아, 나는 이런 스타일이구나"
- 성향에 대한 새로운 인사이트
- 투자 결정 패턴 이해

#### 안전감 (Safety)
- 실제 투자가 아니라는 안심
- 부담 없이 탐색할 수 있는 환경
- 명확한 엔터테인먼트 목적 표시

#### 사회적 연결 (Social Connection)
- 결과를 공유하며 대화 소재 제공
- "너는 어떤 유형이야?" 질문 유발
- 친구들과의 비교 재미

## 5. 차별화 전략

### 5.1 핵심 차별화 포인트

#### 1. 은유 기반 설명 (Metaphor-Driven Explanation)
```
기존 방식:
"삼성전자는 PER 15배, PBR 1.2배입니다."

우리 방식:
"당신의 INTJ 성향에 맞는 '천천히 쌓이는 성' 같은 종목이에요.
단기 변동에 흔들리지 않고 장기적으로 견고하게 성장하는 스타일입니다."
```

#### 2. AI 개인화
- 같은 MBTI라도 투자 경험에 따라 다른 설명
- 사용자 맥락을 반영한 은유 생성
- 매번 새로운 표현 (재방문 유도)

#### 3. 캐릭터 경험
- 투자 = 숫자가 아닌 **캐릭터 스토리**
- "전략가 투자자", "모험가 투자자" 등 정체성 부여
- 감정적 연결 강화

### 5.2 가치 제안 캔버스

#### 고객 작업 (Customer Jobs)
- 투자에 대해 배우고 싶다
- 내 성향을 이해하고 싶다
- 재미있는 콘텐츠를 소비하고 싶다

#### 고통 (Pains)
- 투자 용어가 어렵다
- 어디서부터 시작할지 모르겠다
- 혼자 공부하기 지루하다

#### 이득 (Gains)
- 쉽게 이해할 수 있다
- 나에 대해 알게 된다
- 친구들과 공유할 수 있다

#### 우리의 솔루션
- **Pain Relievers**: 은유로 쉽게 설명, 부담 없는 진입
- **Gain Creators**: 성향 분석, SNS 공유 기능
- **Products & Services**: AI 캐릭터 생성, 맞춤 추천

### 6.1 비즈니스 지표
