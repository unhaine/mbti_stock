## 📌 프로젝트 한 줄 요약

**"MBTI로 알아보는 나의 투자 캐릭터"** - 성격 유형 기반 투자 성향 분석 및 맞춤형 주식 추천 엔터테인먼트 서비스

---

## 🎯 핵심 가치 제안

### 문제 (Problem)
- 주식 투자 서비스는 초보자에게 진입 장벽이 높음 (숫자, 차트, 전문 용어)
- 투자 정보는 많지만, **자기 성향과 연결해 이해할 수 있는 설명**은 부족
- MBTI는 대중적이지만 투자 영역과 연결된 콘텐츠는 거의 없음

### 솔루션 (Solution)
- MBTI 성격 유형을 투자 성향으로 재해석
- AI를 활용한 **은유 기반 설명**으로 복잡한 투자 개념을 쉽게 전달
- 캐릭터화된 결과로 재미와 공유 가능성 극대화

### 차별점 (Differentiation)
- ✅ 주식 종목을 **정보가 아닌 캐릭터 맥락**으로 설명
- ✅ AI 생성 콘텐츠로 개인화된 은유 제공
- ✅ SNS 공유에 최적화된 카드형 UI

---

## 🎨 주요 기능

### 1. MBTI 기반 투자 캐릭터 생성
- 16개 MBTI 유형 선택
- AI가 생성하는 투자 캐릭터 이름 및 설명
- 성향별 투자 스타일 분석 (리스크 성향, 의사결정 방식 등)

### 2. 맞춤형 주식 추천
- MBTI 성향에 맞는 주식 3개 추천
- 각 종목에 대한 **은유적 설명** 제공
  - 예: "불타는 냄비" (고변동성), "천천히 쌓이는 성" (안정 배당주)

### 3. 결과 공유 기능
- 캐릭터 카드 이미지 자동 생성
- SNS 공유 버튼 (카카오톡, 트위터, 페이스북)
- 다른 MBTI 유형 탐색 기능

---

## 🛠️ 기술 스택 (요약)

### Frontend
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **State Management**: Zustand / React Context
- **Image Generation**: html2canvas

### Backend
- **Framework**: FastAPI (Python)
- **AI/LLM**: OpenAI GPT-4 또는 Claude API
- **Data Source**: Yahoo Finance API / KRX Open API

### Infrastructure
- **Frontend Hosting**: Vercel
- **Backend Hosting**: Railway / Render
- **Database**: PostgreSQL (Optional, 초기에는 JSON 파일)

---

## 👥 타겟 사용자

### 주요 타겟
- **연령**: 20~30대
- **특징**: 
  - MBTI에 익숙하고 관심 많은 사용자
  - 주식에 관심은 있지만 전문 지식은 없는 초보자
  - SNS 활동이 활발한 사용자

### 사용 시나리오
1. 심리 테스트처럼 가볍게 접근
2. 결과를 SNS로 공유하며 친구들과 비교
3. 다른 MBTI 유형의 투자 스타일 탐색
4. 투자에 대한 흥미 유발 및 학습 동기 부여
