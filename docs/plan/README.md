## 📋 문서 작성 규칙

### 문서 헤더 형식
모든 문서는 다음 헤더를 포함합니다:

```markdown
# [문서 제목]

> **작성일**: YYYY-MM-DD  
> **최종 수정**: YYYY-MM-DD  
> **상태**: 초안 / 검토중 / 확정  
> **관련 문서**: [링크]
```

### 변경 이력 관리
문서 하단에 변경 이력을 기록합니다:

```markdown
## 📝 변경 이력

| 날짜 | 버전 | 변경 내용 | 작성자 |
|------|------|-----------|--------|
| 2026-01-15 | 0.1 | 초안 작성 | - |
```

### 문서 간 링크
관련 문서는 상대 경로로 링크합니다:

```markdown
자세한 내용은 [03_기술_명세서.md](./03_기술_명세서.md)를 참고하세요.
```
