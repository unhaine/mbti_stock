# MBTI 기반 투자 캐릭터 생성기

## 1. 프로젝트 개요

### 1.1 프로젝트 명
**MBTI 기반 투자 캐릭터 생성기**

### 1.2 프로젝트 목적
본 프로젝트는 MBTI라는 대중적인 성격 유형 지표를 활용하여, 주식 투자라는 복잡하고 부담스러운 영역을 **성향 중심의 캐릭터 경험**으로 재해석하는 AI 서비스이다. 실제 투자 수익을 예측하거나 권유하는 것이 아니라, 사용자의 성향에 맞는 투자 캐릭터와 종목 유형을 재미있고 이해하기 쉬운 방식으로 제시하는 것을 목표로 한다.

### 1.3 프로젝트 성격
- AI 서비스 개발 자기계발 수업용 프로젝트
- 엔터테인먼트 + 성향 분석 중심
- 투자 권유 목적이 아닌 **의사결정 이해 보조 실험**

---

## 2. 기획 배경 및 문제 정의

### 2.1 배경
- 주식·투자 서비스는 초보자에게 진입 장벽이 높음
- 숫자, 차트, 지표 중심 UI로 인해 흥미 저하
- MBTI는 이미 대중적으로 친숙한 자기 이해 도구로 자리 잡음

### 2.2 문제 정의
> 투자 정보는 많지만, 사용자가 **자기 성향과 연결해 이해할 수 있는 설명**은 부족하다.

---

## 3. 서비스 핵심 콘셉트

### 3.1 핵심 키워드
- MBTI
- 투자 성향
- 캐릭터화
- 은유적 설명
- AI 생성 콘텐츠

### 3.2 서비스 한 줄 설명
> “MBTI로 알아보는 나의 투자 캐릭터”

### 3.3 차별점
- 주식 종목을 **정보가 아닌 캐릭터 맥락**으로 설명
- AI를 활용한 성향 기반 문구·은유 생성
- 결과 공유에 최적화된 카드형 UI

---

## 4. 타겟 사용자

### 4.1 주요 타겟
- 20~30대
- MBTI에 익숙한 사용자
- 주식에 관심은 있지만 전문 지식은 없는 사용자

### 4.2 사용 시나리오
- 심리 테스트처럼 가볍게 사용
- 결과를 SNS로 공유
- 다른 MBTI 유형도 탐색

---

## 5. 사용자 흐름 (User Flow)

1. 랜딩 페이지 접속
2. 서비스 소개 및 온보딩
3. MBTI 선택
4. (선택) 투자 경험 입력
5. AI 캐릭터 생성 로딩
6. 투자 캐릭터 결과 확인
7. 추천 주식 카드 탐색
8. 결과 공유 또는 재탐색

---

## 6. 핵심 기능 정의

### 6.1 MBTI 입력
- 16개 MBTI 중 선택
- 별도 테스트 제공하지 않음 (간결성 유지)

### 6.2 투자 캐릭터 생성 (AI)
- MBTI 기반 투자 성향 해석
- 캐릭터 이름 및 한 줄 정의 생성

### 6.3 추천 주식 카드 UI
- 카드형 추천 UI
- 종목 + 은유적 태그 제공
- 클릭 시 추천 이유 확장 표시

### 6.4 은유 기반 추천 설명
- 예: “불타는 냄비”, “천천히 쌓이는 성”, “롤러코스터” 등
- 성향과 주식 특성을 연결한 설명 제공

### 6.5 결과 공유
- 캐릭터 카드 이미지 형태 공유
- SNS 친화적 문구 자동 생성

---

## 7. AI 활용 방식

### 7.1 AI 역할
- 투자 캐릭터 설명 생성
- 추천 주식에 대한 은유적 설명 생성
- MBTI 성향에 맞는 말투 조절

### 7.2 AI 사용 원칙
- 실제 투자 판단 또는 수익률 예측 금지
- 설명 중심, 해석 중심 출력

---

## 8. UI/UX 설계 방향

### 8.1 디자인 원칙
- 주식 앱이 아닌 **캐릭터 테스트 느낌**
- 차트·가격 정보 최소화
- 카드형 UI 중심

### 8.2 추천 UI 구조
- 캐릭터 요약 카드
- 추천 주식 카드 3개
- 카드 확장 시 설명 노출

---

## 9. 기술 스택 (예시)

- Frontend: React / Next.js
- Backend: FastAPI (Python)
- AI: LLM API 활용
- 데이터: 더미 종목 데이터 또는 공개 API

## 10. 기대 효과

- 투자에 대한 심리적 장벽 완화
- AI 설명 가능성(Explainability) 실험
- 성향 기반 UX 설계 경험 축적

## 11. 한계 및 유의사항

- 실제 투자 조언 아님
- MBTI의 과학적 한계 인지 필요
- 엔터테인먼트 목적 명확화

---

## 12. 향후 확장 아이디어

- MBTI별 가상 포트폴리오 시뮬레이션
- 시장 상황별 캐릭터 반응
- 다른 성향 지표(Big5 등) 적용

---

## 13. 결론

본 프로젝트는 AI를 활용해 사용자의 성향을 이해하고, 이를 투자라는 복잡한 영역에 **재미있고 직관적인 캐릭터 경험**으로 연결하는 실험적 서비스이다. 기술적 완성도보다도, AI가 어떻게 사용자의 의사결정을 이해 가능하게 돕는지를 탐구하는 데 의의가 있다.
